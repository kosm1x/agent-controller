## The defining constraint

Per the Spine 7 spec: the tools MUST NOT execute a git mutation without operator confirmation. `close-phase` writes only `*.draft` files; `audit-closure`/`resume` only READ git (`git cat-file -e`, `git log`). **Auditor verified this holds under adversarial testing** — path-traversal (the phase-arg regex `^[a-z0-9.]+$` admits `..` but no `/`, so no directory escape), command-injection (commit hashes are `[0-9a-f]`-only — no shell metacharacter can reach `execSync`), and exit-code propagation.
