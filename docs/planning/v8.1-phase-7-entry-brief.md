## Phase 7 ‚Äî what to build (spec ¬ß12 Phase 7, ~2 days)

1. **N-turn counter** into the dispatcher ‚Äî after every Nth foreground task completes (`reflection_freq=5`), fire-and-forget a reflection via `safe_create_task` (¬ß6 Trigger 1).
2. **Cron 06:00 morning surface** ‚Äî replaces the existing `rituals/morning.ts`; pulls the latest pending `proposed_briefings` morning row, or generates one synchronously if none (¬ß6 Trigger 2).
3. **Idle-detect job** ‚Äî 4h no foreground activity AND an active project has a stalled task ‚Üí fire a stalled-focused reflection (¬ß6 Trigger 3).
4. **Per-surface throttling** ‚Äî idle-detect max 1/12h; cron supersede so no stack of unread briefs.
5. Tests: triggers fire at the correct boundaries, throttling holds.

The design detail is in spec ¬ß6 (verbatim trigger semantics). Each trigger calls `constructBriefing({surface, cursorName, trigger})` and/or `runReflection(...)` ‚Äî both already built.

## Load-bearing entry conditions

- **Re-evaluate seed sufficiency BEFORE enabling an operator-facing trigger** (closure audit W1). The Phase A item-3 gate was deferred to "live-retrieval time" ‚Äî which is Phase 7. `general_events` has only 12 seed rows; auto-discovery (`S4-B1-W4`) never landed. Decide at Phase 7 entry: is 12 enough for a real morning brief, or does auto-discovery / a richer seed need to land first? Do not enable a brief operators see on top of an under-populated retrieval layer.
- **Anti-mission: no autopilot.** Triggers generate `proposed_briefings` rows with `status='pending'`. They must NOT auto-deliver or auto-act ‚Äî delivery + promote/discard is Phase 8. A Phase 7 trigger's job ends at a persisted pending briefing.
- **Verify-live per bundle** ‚Äî every Conway/V8.1 phase shipped a bug only the first live run caught. A trigger that fires (or fails to fire) at the wrong boundary is exactly that class.
- **`constructBriefing` runs real inference** once triggered ‚Äî Phase 7 is the first phase with live LLM cost on this path. Budget for it; the N-turn trigger is lazy (only fires on foreground activity) by design.
